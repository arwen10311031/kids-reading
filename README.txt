書籍管理系統 (Book Collection Manager)
=====================================

檔案位置: 書籍管理.html
開啟方式: 用瀏覽器開啟此HTML檔案

功能特色:
=========

1. 書庫 (Library Tab)
   • 顯示所有516本書籍
   • 搜尋功能 (依書名、出版社、備註)
   • 篩選器:
     - 全部: 顯示全部書籍
     - 已讀: 只顯示已閱讀的書
     - 未讀: 只顯示未閱讀的書
     - 棄讀: 只顯示放棄的書
     - 已售: 只顯示已售出的書
   • 排序選項:
     - 購買日期 (新→舊/舊→新)
     - 價格 (高→低/低→高)
     - 書名 (A→Z)
   • 點擊書卡編輯或刪除

2. 新增 (Add Tab)
   • 完整表單添加新書籍
   • 自動重複檢查 (輸入書名時提示)
   • 支援所有字段:
     - 書名、出版社
     - 購買日期、價格、購買平台
     - 閱讀狀態、閱讀日期
     - 售出價和售出日期
     - 備註

3. 統計 (Statistics Tab)
   • 總藏書數和總投資額
   • 閱讀進度條 (已讀/總數)
   • 閱讀狀態分佈
   • 出版社排行 (Top 5)
   • 購買平台分佈 (Top 5)
   • 購買趨勢圖表 (近12個月)

4. 設定 (Settings Tab)
   • 匯出資料: 下載 JSON 備份檔
   • 匯入資料: 從 JSON 檔案還原
   • 清除所有資料: 重置應用 (危險操作)

數據儲存:
==========
• 所有數據存儲在瀏覽器的 localStorage
• 無需網路連接，完全離線使用
• 數據永久保存直到手動清除

使用提示:
=========
• 第一次開啟時自動載入516本書籍數據
• 每次編輯或新增都自動保存
• 建議定期匯出數據備份
• 刪除操作需確認，可避免誤刪

顏色代碼:
=========
✅ 綠色: 已閱讀
❌ 紅色: 棄讀
❓ 灰色: 未讀
💛 黃色: 已售出

技術詳情:
=========
• 純 HTML5 + CSS3 + JavaScript
• 無需任何外部依賴
• 響應式設計，適合手機使用
• 最小螢幕寬度: 360px

問題排除:
=========
若數據丟失:
1. 檢查瀏覽器是否允許 localStorage
2. 嘗試使用隱私模式/無痕模式
3. 從備份的 JSON 檔案匯入

若顯示不正常:
1. 清除瀏覽器快取
2. 使用最新版本的瀏覽器
3. 嘗試其他瀏覽器 (Chrome, Firefox, Safari, Edge)

祝您使用愉快！📚
