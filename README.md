# Kids Reading - 兒童閱讀管理系統

一個為家長設計的手機版兒童書籍管理與閱讀獎勵系統，前端單檔 + Google Sheets 雲端同步，多裝置即時共用。

**線上版：** https://arwen10311031.github.io/kids-reading/

## 解決什麼問題

1. **避免重複購書** — 累積數百本書後很難記得買過什麼，搜尋和重複檢查讓你輸入書名就能馬上比對
2. **二手書售價參考** — 記錄每本書的買入價與售出價，清楚掌握成本和獲利
3. **追蹤小孩閱讀** — 記錄閱讀日期與數量，按月/年統計閱讀狀況
4. **閱讀獎勵機制** — 遊戲化的每日打卡積分系統，讓小孩有動力持續閱讀
5. **多裝置同步** — 資料存在 Google Sheets，手機、電腦都能用同一份資料

## 功能總覽

### 書庫管理

- 書卡列表顯示所有書籍，清楚標示書名、出版社、買入價、購買日、閱讀日、售出資訊
- 多層篩選：文字搜尋 × 出版社 × 閱讀狀態（已讀/未讀/棄讀） × 售出狀態（已售/未售）
- 排序：購買日新舊、價格高低、書名
- 點擊書卡可編輯或刪除

### 新增書籍

- 完整表單：書名、出版社（下拉）、購買平台（下拉）、買入價、購買日、閱讀狀態、售出資訊、備註
- 輸入書名時即時比對現有書籍，提醒可能的重複

### 統計報表

- 財務摘要：購買總金額 / 已售出金額 / 淨成本
- 閱讀進度：已讀數量與閱讀率
- 每月閱讀數量長條圖（可點擊展開明細）
- 每年閱讀數量統計
- 出版社排行 Top 5
- 購買平台分佈 Top 5

### 獎勵系統（小孩打卡）

每日任務打卡，讀完書按 `+` 即時累計積分：

| 任務 | 預設積分 | 說明 |
|------|---------|------|
| 指定中文 1 本 | 5 點 | 家長指定的中文書 |
| 隨意中文 1 本 | 3 點 | 自由選讀的中文書 |
| 英文朗讀 1 本 | 5 點 | 英文書朗讀練習 |
| 英文聽 2 本 | 1 點 | 聽英文有聲書 |
| 英文閱讀 1 本 | 1 點 | 英文書閱讀 |
| 早起/ICRT/其他 | 1 點 | 早起聽 ICRT 等 |

**小孩 / 家長雙模式：** 小孩提交打卡（待確認），家長輸入 4 位 PIN 碼進入家長模式審核通過後才計入積分。PIN 輸入以 `••••` 遮罩顯示。

**每月權重調整：** 家長可以在設定頁調整各月的積分權重。想加強哪個項目就把分數調高，小孩自然會為了高積分多做那個任務。

**等級成就系統：**

| 等級 | 累積積分 | 稱號 |
|------|---------|------|
| 🌱 | 0 – 100 | 小書蟲 |
| 🌿 | 101 – 300 | 閱讀新星 |
| 🌳 | 301 – 600 | 知識探險家 |
| 🏆 | 601 – 1,000 | 閱讀達人 |
| 👑 | 1,001 – 1,500 | 書香大師 |
| 🌟 | 1,500+ | 超級閱讀王 |

**其他功能：** 本月日曆活躍度、歷史月份查詢、智慧鼓勵語

### 訪客模式

- 預設為訪客模式（唯讀），金額顯示為 `*****` 保護隱私
- URL 加上 `?owner` 參數進入管理員模式
- 程式層面 14 個函式加上 `ownerOnly()` 守衛，防止 console 繞過

### 設定

- JSON 匯出/匯入（書籍資料與獎勵資料分開備份）
- 新增/刪除任務類型
- 每月積分權重設定（可複製上月再微調）
- 家長 PIN 密碼設定
- 清除全部資料

## 技術架構

- 前端：純 HTML5 + CSS3 + JavaScript，單一檔案
- 後端：Google Apps Script（免費 API）
- 資料庫：Google Sheets（書單、獎勵、待確認、設定 四個工作表）
- 部署：GitHub Pages
- 響應式設計，針對手機螢幕（360–480px）最佳化
- 圖表以純 CSS 繪製（進度條、長條圖）
- 觸控友善：最小點擊區域 44px

### 資料流

```
瀏覽器 (書籍管理.html)
  ↕ fetch API
Google Apps Script (Web App)
  ↕ SpreadsheetApp
Google Sheets (資料庫)
```

## Google Sheets 工作表結構

| 工作表 | 用途 |
|--------|------|
| 書單 | 書籍資料（id、書名、出版社、價格、閱讀狀態...） |
| 獎勵 | 每月已確認的打卡紀錄（月份 × 任務 × 每日次數） |
| 待確認 | 小孩提交但尚未審核的打卡紀錄 |
| 設定 | 系統設定（家長 PIN 等） |

## 使用方式

1. 用手機或電腦的瀏覽器開啟線上版或本地 `書籍管理.html`
2. 頁面載入時自動從 Google Sheets 讀取最新資料
3. 所有新增、編輯、刪除操作即時同步回 Google Sheets
4. 訪客模式預設唯讀，管理員在網址加 `?owner` 即可完整操作

如果想先了解獎勵系統的使用流程，可以開啟 `獎勵Demo.html` 跟著互動式教學操作。
